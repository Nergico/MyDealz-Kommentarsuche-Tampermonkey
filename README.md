**MyDealz Kommentar-Volltextsuche (Tampermonkey Edition)**
Dieses Skript rüstet eine extrem praktische Funktion für MyDealz nach: Eine echte Volltextsuche für Kommentare innerhalb eines Deals oder einer Diskussion.  
Wer MyDealz kennt, weiß: Bei beliebten Deals mit hunderten Antworten dauert es oft ewig, bestimmte Informationen zu finden. Dieses Skript löst das Problem, indem es alle Kommentare ausliest, in einem lokalen Cache zwischenspeichert und blitzschnell durchsuchbar macht.  
Vorteil der Tampermonkey-Version: Im Gegensatz zur Lesezeichen-Variante ist das Skript fest in deinem Browser integriert und steht dir auf jeder Deal-Seite sofort zur Verfügung, ohne dass du erst ein Lesezeichen anklicken musst.  

**Du suchst die Version ohne Browser-Erweiterung? > Hier geht es zum Repository für die Lesezeichen / Bookmarklet Variante:** https://github.com/Nergico/MyDealz-Kommentarsuche-mit-Lesezeichen  

** Die 2 Varianten (Wähle deine bevorzugte Version)**  
Ich biete hier zwei verschiedene Ansätze an. Wähle die Version, die besser zu deinem Workflow passt.  

**🔹 Variante 1: Pro-Suche (Modern & Inline)**  
Dies ist die aktuellste und umfangreichste Version. Sie fügt neben der Kommentar-Sortierung einen Button "🔍 Pro-Suche starten" ein. Sobald du darauf klickst, wird die Suche geladen.  
Besonderheit: Bietet schnelle Klick-Filter (z. B. "Letzte 24h", "1+ Reactions") und eine extrem übersichtliche Mini-Ansicht für gefundene Kommentare.  

**🔹 Variante 2: Tab-Suche (Ergebnisse im neuen Fenster)**  
Diese Version rüstet eine klassische Suchleiste direkt über den Kommentaren nach. Sobald du einen Begriff eingibst und auf Suchen klickst, öffnet sich ein separater Tab/Fenster mit den Ergebnissen.  
Besonderheit: Die MyDealz-Seite selbst bleibt übersichtlich, da die eigentliche Suche und die Ergebnisse in einem eigenen Bereich stattfinden.  

**🛠️ Installation**  
Lade dir die kostenlose Browser-Erweiterung Tampermonkey herunter (für Chrome, Firefox, Edge oder Safari).  
Stelle sicher, dass in den Erweiterungs-Optionen deines Browsers die Berechtigung "Benutzerskripts zulassen" (Entwicklermodus) aktiviert ist.  
Klicke auf das Tampermonkey-Icon in deiner Browserleiste und wähle "Neues Skript erstellen".  
Kopiere den Code deiner gewünschten Variante (siehe oben) komplett in den Editor.  
Gehe auf Datei > Speichern. Fertig!  

**✨ Features & Wichtige Hinweise**  
Lokaler Cache: Beim ersten Start eines Deals werden die Kommentare heruntergeladen. Jede weitere Suche im selben Deal passiert danach in Millisekunden. Bei sehr vielen Kommentaren kann der erste Ladevorgang einen Moment dauern.  
Dark Mode Support: Das Design passt sich automatisch an dein MyDealz-Theme (Hell oder Dunkel) an.  
Wichtig zu Variante 2 (Pop-ups): Da sich die Suche in einem neuen Fenster öffnet, muss dein Browser Pop-ups für www.mydealz.de zulassen.  
Immer aktuell: Das Skript erkennt automatisch, wenn neue Kommentare auf MyDealz gepostet wurden, und bietet dir an, diese per Knopfdruck nachzuladen.
